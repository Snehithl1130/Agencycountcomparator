# Agencycountcomparator

A web-based dashboard to compare agency agent counts between two different days using JSON data.  
Designed for quick analysis, reporting, and visualization of workforce or operational changes.


## Features
- Search and filter agencies instantly  
- Download comparison results as CSV  
- Dark and light theme toggle  
- Difference table sorted by change  
- Top 20 changes chart visualization  
- Fast JSON comparison engine  
- Clean and responsive dashboard UI  


## Use Case

Paste agent count JSON from:

- Day 1 report  
- Day 2 report  

Click **Compare Now** to instantly view:

- Increased agencies  
- Decreased agencies  
- No change  
- Top movers chart  
- Exportable CSV report  

Useful for:

- Operations teams  
- Insurance reporting  
- Workforce tracking  
- Daily SLA monitoring  
- Data comparison tasks
